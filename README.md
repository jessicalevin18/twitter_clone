# Twitter Clone
[![Tests](https://github.com/jessicalevin18/twitter_clone/actions/workflows/start_services.yml/badge.svg)](https://github.com/jessicalevin18/twitter_clone/actions/workflows/start_services.yml)

A fully functional database-backed webpage created integrating knowledge of Python, Flask, HTML/CSS, Jinja2, and SQLite3.

## Features
- CRUD (Create, Read, Update, Delete) functionality
- User authentication and authorization
- Search functionality with full-text search and relevancy ordering
- Database schema design with indexing for optimal performance

## Technologies Used
- Python 3.12.3
- Flask
- HTML/CSS
- Jinja2
- SQLite3
- Docker

## Project Structure
- Development and production Docker Compose files
- GitHub repository with automated testing and deployment
- Database schema and test data loading scripts
